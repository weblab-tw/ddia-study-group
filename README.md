## Rules

1. 共分為三期，每期範圍為中文翻譯 Repo 的一個部分

3. 每期中間隔 2-4 週，每一期結束會舉辦慶功宴或是公開分享會作為結尾。
4. 主持人： KK & Kyle (徵求自願者)
